# Representaciones Hidráulicas J.E. - Web Management System

A comprehensive web application developed for **Representaciones Hidráulicas J.E. S.A.S.** to digitalize and optimize the management, control, and maintenance of hydraulic systems and mechanized equipment.

This project replaces manual processes with an integrated platform that improves operational efficiency, work traceability, customer communication, and maintenance management. The system was developed following the **Extreme Programming (XP)** methodology to ensure continuous delivery, testing, and client feedback throughout the development process. :contentReference[oaicite:0]{index=0}

## 📖 About the Project

Representaciones Hidráulicas J.E. specializes in the repair, manufacturing, and maintenance of hydraulic systems. Before this project, most operational processes were managed manually through spreadsheets, paper documents, and messaging applications.

The objective of this application is to centralize company operations into a secure and scalable web platform that allows administrators, technicians, and clients to interact efficiently from anywhere.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/yourrepository.git
```

Move the project into your web server directory.

For XAMPP:

```
htdocs/
```

Import the MySQL database.

Configure the database credentials in:

```
config/database.php
```

Start Apache and MySQL.

Open the browser:

```
http://localhost/project-name
```
